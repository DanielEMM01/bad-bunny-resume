# Bad Bunny – Artist Resume (HTML)

This project is a single-page HTML resume created for **CPRG 213 – Web Development** 

The webpage presents a resume-style profile for the artist **Bad Bunny**, using semantic HTML elements and structured content to meet assignment requirements as a resume.

---

## Project Requirements Met
- Semantic HTML5 structure
- One main heading and multiple sections
- Image with alternative text
- External link
- Information displayed using an HTML table
- Clean document outline

---

## Technologies Used
- HTML5
- GitHub Pages
